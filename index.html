Meet Gagster Agent, the newest grumpy AI Artist Agent which brings an entire AI Framework for creating the funniest memes and cutting-edge social commentary on the web.

What makes ACC different from other automations?

Art feedback ingestion

  • Gagster Agent is constantly ingesting feedbacks from community on X and Telegram
  • The models are constantly adapting and evolving by processing the ingested data
  • Next art created will reflect community sentiment in some way

Unique art concept generation

Inspired on the infinite backrooms, we have multiple models running on the platform and each artwork generated is the result of a brainstorming session between the models.

Three main topics are discussed by the models throughout an art generation:

  • Scene: The background or location for the piece
  • Characters: Who will be involved in the scene
  • Context: The situation or motivation giving sense to the scene

The Concept Models

  • Claude 3.5 Haiku
  • Llama 3.3
  • GPT-4o

ACC-OS Custom RAG model

All the decision making is made by our own custom model based on llama-3.3.

This model leverages the brainstorming session results and decide which collaborations from each model will be used in the final composition.

If you keep an eye on the Act 3 activity session you might be able to spot some logs related to the creation process.

Gagster Framework is the origin, initial, code base of the ACC Framework.

Gagster is open source and free to use, modify, transform and share. Join the revolution and create your own smart meme agent.

Powered by AAC, this project is an open source initiative part of the Agentic Creative Complex framework to empower memes, by automating smart meme agents to generate art and share it to whole communities on X and Telegram. The agentic smart meme can think by itself, talk, learn and remember its users, use generative AI to create its own art and leverage integrations to share it to the world.


Check our repository on Github to learn more about the project and how you use it to create your own agents and contribute to it.


Versions:

Gagster comes in two main versions:

  • Community: Open source limited version that contains all core features, but a single agent process requests through OpenAI. Meme lovers from everywhere are invited to collaborate and improve on this framework.
  • ACC: This version is currently the core version (ACC-OS v2.0.4) running on official Gagster website and socials. This version leverages three agents - OpenAI gpt-4o, custom Llama 3.3 and custom Claude 3.5 Haiku models, analyzing community art feedback and requests and discussing with each other to create its next art piece concept.
